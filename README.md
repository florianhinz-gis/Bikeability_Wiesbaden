## Bicycle Friendliness in Wiesbaden 

This project presents an interactive web map that visualizes the spatial distribution of bicycle friendliness in Wiesbaden.

The analysis is based on a multi-criteria index developed by the German Aerospace Center (DLR), combining factors such as:
- cycling infrastructure on main streets
- crossroad density
- share of small streets
- green spaces
- share and repair facility density

Higher values indicate better conditions for cycling.

The map allows users to explore local variations and understand which urban characteristics contribute to a more bike-friendly environment.

More information about the index:
https://www.dlr.de/de/vf/aktuelles/nachrichten/bikeability-tool-fahrradfreundlichkeit
